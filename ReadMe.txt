Publication: "A software tool to visualize soil moisture dynamics of an irregular-shaped profile"
by Jan G. Wesseling, Klaas Oostindie, Coen J. Ritsema  and Louis W. Dekker.

Published in Computers and Geosciences.

This repository contains the following items:
- Source code of the program
- User manual
- Examples of input
- A movie made with the program showing the measured soil moisture content
  in a potato field made with the program TDRfree.
